Summary

Saucer sling is a 2D puzzle game where you have to maniplate the gravitational pull of planets to slingshot your spaceship to the end of each level. This project was developed in a group of 6. This was my first Unity project and so much of my time on the project was spent getting familiar with the Unity interface.

Want to play it? The itch.io page can be found here.
The GGJ 2018 page for the game can be found here.

What did I do?

In this project I designed several levels. My was to try and box out the "hook" of each level early so that the concept could still be used without needing the gravity mechanics to have been fully implemented.
