Project information

  • Project Type: Game Jam
  • Start Date: 28 January, 2019
  • Finish Date: 31 January, 2019
  • Tools: Unity, C#, Constellate API

Summary

Roast royale is a local party game where 4 Players compete to create a roast dinner from the family dinner table by using a controller to move their arm. This silly little party game evokes feelings of a hectic family dinner to try and capture a small piece of the essence of Home. It was developed as part of the Global Game Jam 2019 where the theme was "what Home means to you". This project was developed in a team of 6.

Want to play it? The itch.io page can be found here.
The GGJ 2019 page for the game can be found here.

What did I do?

I implemented the audio-handling system. This transitions to different music states when the game is in the menu, in game or after the game, and pools audio sources during gameplay so multiple sounds are not instantiated in a short time.
